# Podwojone PDF i MD

Prosta aplikacja konsolowa w C#, która wyszukuje pliki o tej samej nazwie,
różniące się jedynie rozszerzeniem **.pdf** oraz **.md**.

Program wyświetla listę znalezionych par w terminalu, a następnie – po
potwierdzeniu przez użytkownika – usuwa pliki **.pdf**, pozostawiając tylko
pliki **.md**.

---

## Działanie

- skanowanie folderu `pliki`
- wykrywanie par `.pdf` + `.md`
- prezentacja wyników w czytelnej tabeli
- usuwanie plików `.pdf` po zatwierdzeniu
